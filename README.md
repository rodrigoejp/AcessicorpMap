# AcessicorpMap
Implementação de aplicativo Android [Java] para apresentação de TCC de Rodrigo Egidio Joenk Pereira


Inclui a utilização das seguintes tecnologias, mas não limitado:
- Googlemaps
- Geocoding
- GPS
- BLE

Work in progress:

- [ ] Beacon
